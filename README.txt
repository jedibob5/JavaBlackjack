A personal coding project which plays blackjack in the Java console.

Plays a full game of blackjack, including the player's hand and the dealer's hand.

Includes tracking of which cards in the deck have been played during the round,
ensuring that 5 aces can't be played, etc. Only features missing are any related
to betting, there is no score kept in between individual rounds. This includes
doubling down or splitting, though all other blackjack functions are present.
